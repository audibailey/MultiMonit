# MultiMonit
A M/Monit Alternative coded in Python.

Infomation:
Python 2.7+

Required Modules:
Jinja2 >= 2.8
CherryPy >= 4.0.0
MarkupSafe >= 0.23

Instructions:
Run the run.py and goto localhost:3005 for the setup to begin.
